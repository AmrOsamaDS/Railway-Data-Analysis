## Project Overview
This project focuses on analyzing United Kingdom railway data to extract meaningful insights related to passenger traffic, station performance, and travel trends. The goal is to support data-driven decision-making and improve transportation planning and operational efficiency.

## Objectives
- Analyze passenger volume trends across different stations.
- Identify peak travel periods.
- Evaluate station performance.
- Discover patterns in train usage and demand.

## Tools & Technologies
- Power BI (Data Visualization & Dashboarding)
- Microsoft Excel (Data Cleaning & Preprocessing)


## Dataset Description
The dataset contains railway-related data including:
- Station names
- Passenger counts
- Travel dates and times
- Routes information
- Ticket and journey details

## Data Processing Steps
1. Data cleaning and handling missing values.
2. Data transformation and normalization.
3. Creating calculated columns and measures.
4. Aggregating passenger and station-level metrics.
5. Preparing data model for visualization.

## Key Insights
- Identification of the busiest railway stations.
- Monthly and yearly passenger trends.
- Peak travel hours and high-demand routes.
- Comparison between different regions and stations.

## How To Use This Project
1. Download or clone the repository.
2. Open the Power BI file from the `dashboard` folder.
3. Make sure the dataset path is correctly linked.
4. Refresh the data.
5. Explore the interactive dashboard.

## Results & Impact
This analysis provides valuable insights that can help transportation authorities and planners to:
- Improve service scheduling.
- Optimize station operations.
- Enhance passenger experience.
- Support strategic transportation decisions.
